## Fixed
- Files with "special" characters in their names (non-alphanumeric, dash, underscore, dot) will no longer fail to be sent in embeds/components
- Addressed an issue where triggers would have significant update delay after modifying their flows via the site
- Fixed an issue where duplicated messages could not be selected independently when choosing messages to send
- Fixed hang when trying to send a link button with an invalid URL
